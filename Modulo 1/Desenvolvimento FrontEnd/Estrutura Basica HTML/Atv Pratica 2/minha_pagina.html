Segunda Atividade Prática

Essa atividade consistiu na criação e estruturação de uma página na web utilizando HTML básico.

Para realizar essa atividade eu tive que aplicar meus conhecimentos básicos de HTML para criar e estruturar uma página web simples, inserindo títulos, subtítulos, parágrafos e links, além de aplicar conceitos de semânticas empregando as tags para criar uma seção (section) e um rodapé (footer).
